Robotic Arm



The main plan for this is to use a Stewart Platform stacked ontop of another Stewart Platform,

these two platforms together should provide not only the degrees of motion required but also the range of motion required.

The software for this project is almost in a production state


The three checks

  • We already are in possession of most of the motors required to create this so cost passes
  • As we are in possession of most of the materials required it is very much possibile to make
  • As far as we are aware there are no laws against small robotic arms so it passes

As it has passed all the tests, once the software is production ready the robot can be built
